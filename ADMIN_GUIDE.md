## 📊 Fonctionnalités du Dashboard

### 1. Tableau de Bord
- **Statistiques en temps réel** : Utilisateurs, cours, matières
- **Graphiques d'activité** : Croissance des utilisateurs, matières populaires
- **Actions rapides** : Accès direct aux fonctions principales

### 2. Gestion des Utilisateurs
- **Voir tous les utilisateurs** : Liste complète avec rôles
- **Modifier les profils** : Changer les rôles (Étudiant/Enseignant/Admin)
- **Supprimer des comptes** : Gestion complète des utilisateurs
- **Statistiques** : Nombre d'utilisateurs par rôle

### 3. Gestion des Matières
- **Ajouter des matières** : Créer de nouvelles matières avec couleurs
- **Modifier les matières** : Éditer nom, description, couleur
- **Supprimer des matières** : Gestion complète du contenu
- **Organisation** : Système de slugs automatiques

### 4. Gestion des Cours
- **Créer des cours** : Ajouter du contenu éducatif
- **Associer aux matières** : Lier les cours aux matières appropriées
- **Niveaux de difficulté** : Débutant, Intermédiaire, Avancé
- **Contenu riche** : Description et contenu détaillé

### 5. Paramètres du Site
- **Informations générales** : Nom, description, contact
- **Maintenance** : Cache, sauvegardes
- **Configuration** : Paramètres globaux

## 🔧 Utilisation

### Ajouter une Matière
1. Aller dans "Matières"
2. Cliquer sur "Ajouter"
3. Remplir : Nom, Description, Couleur
4. Sauvegarder

### Créer un Cours
1. Aller dans "Cours"
2. Cliquer sur "Ajouter"
3. Remplir : Titre, Matière, Niveau, Description, Contenu
4. Sauvegarder

### Gérer les Utilisateurs
1. Aller dans "Utilisateurs"
2. Voir la liste complète
3. Cliquer sur "Modifier" pour changer le rôle
4. Cliquer sur "Supprimer" pour supprimer un compte
